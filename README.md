Adminisztrációs felület (WPF) használata
A WPF alapú adminisztrációs alkalmazás indítására és telepítésére több lehetőség áll
rendelkezésre a környezettől függően.
Indítás fejlesztői környezetből (Visual Studio)
– A projekt .sln fájljának megnyitása a Visual Studio alkalmazásban
– A WPF projekt beállítása Startup Project-ként
– Az alkalmazás indítása a Start gombbal vagy az F5 billentyű segítségével

Telepítés telepítőcsomag segítségével
A telepítőcsomag (.exe vagy .msi) a PhoneScoutAdminInstaller/Debug mappában
található.
– A telepítő futtatásával a varázsló végigvezeti a felhasználót a telepítési folyamaton
– A telepítést követően az alkalmazás regisztrálásra kerül a rendszerben
– Az alkalmazás az API szerver futása esetén azonnal használatba vehető
